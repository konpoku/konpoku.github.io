特征线法

常系数运输方程初值问题

对于方程

∂tρ+a∂xρ=0,x∈R,t>0\partial_t \rho + a \partial_x \rho = 0, x \in \mathbb{R}, t > 0 ∂t​ρ+a∂x​ρ=0,x∈R,t>0

给定初值条件

ρ(x,0)=ρ0(x)\rho(x,0) = \rho_0(x) ρ(x,0)=ρ0​(x)

我们发现在特征线上，ρ\rhoρ 保持不变，然后通过特征线反推每一点ρ\rhoρ的值
具体而言是这样的，特征线是下面的常微分方程初值问题的解

dxdt=a,x(0)=x0\frac{\rm{d}x}{\rm{d}t} = a, x(0) = x_0 dtdx​=a,x(0)=x0​

其实这条线就是x(t,x0)=at+x0x(t,x_0) = at + x_0x(t,x0​)=at+x0​，但是写成微分形式其实是为了下面的推导
根据全微分公式

df=∂f∂xdx+∂f∂tdt\rm{d}f = \frac{\partial f}{\partial x} \rm{d}x + \frac{\partial f}{\partial t} \rm{d}t df=∂x∂f​dx+∂t∂f​dt

我们有

dρ=∂xρdx+∂tρdt\rm{d}\rho = \partial_x \rho \rm{d}x + \partial_t \rho \rm{d}t dρ=∂x​ρdx+∂t​ρdt

dρdt=∂xρdxdt+∂tρ=∂tρ+a∂xρ=0\frac{\rm{d}\rho}{\rm{d}t} = \partial_x\rho \frac{\rm{d}x}{\rm{d}t} + \partial_t \rho = \partial_t \rho + a\partial_x \rho = 0 dtdρ​=∂x​ρdtdx​+∂t​ρ=∂t​ρ+a∂x​ρ=0

最后一步推导使用了原方程条件
上面的推导表明在特征线上ρ\rhoρ为常数，因此对于(x,t)(x,t)(x,t)平面上的任何一个点(x0,t0)(x_0,t_0)(x0​,t0​)，我们都可以找到一条过该点的的特征线x(t,x0−at0)=a(t−t0)+x0x(t,x_0 - at_0) = a(t-t_0) + x_0x(t,x0​−at0​)=a(t−t0​)+x0​，这条线与t=0t=0t=0有一交点x=x0−at0x=x_0-at_0x=x0​−at0​，在该点ρ\rhoρ的值是确定的（初值条件），有ρ(x0−at0,0)=ρ0(x0−at0)\rho(x_0-at_0,0) = \rho_0(x_0-at_0)ρ(x0​−at0​,0)=ρ0​(x0​−at0​)
实际上我们就推导出了ρ(x,t)=ρ0(x−at)\rho(x,t) = \rho_0(x-at)ρ(x,t)=ρ0​(x−at)

值得注意的事情是，我们不妨画出t=0t=0t=0的时候ρ\rhoρ随xxx变化的图像，那么ttt变大即图像右移，ttt减小即图像左移，因此被称为输运方程

变系数输运方程初值问题

现在来讨论更为复杂的情况

∂tρ+∂x(v(x)ρ)=0,ρ(x,0)=ρ0(x),x∈R,t>0\partial_t \rho + \partial_x(v(x)\rho) = 0, \rho(x,0) = \rho_0(x), x\in \mathbb{R}, t > 0 ∂t​ρ+∂x​(v(x)ρ)=0,ρ(x,0)=ρ0​(x),x∈R,t>0

经过数学家不知道多少年的研究我们知道这个方程的特征线长这样

dxdt=v(x(t)),x(0)=x0\frac{\rm{d}x}{\rm{d}t} = v(x(t)), x(0) = x_0 dtdx​=v(x(t)),x(0)=x0​

然后我们把方程稍微变形一下，把变系数拿出来

∂tρ+v(x)∂xρ=−v′(x)ρ\partial_t \rho + v(x)\partial_x \rho = -v'(x)\rho ∂t​ρ+v(x)∂x​ρ=−v′(x)ρ

同样对ρ\rhoρ求全微分

dρdt=∂tρ+dxdt∂xρ=∂tρ+v(x)∂xρ=−v′(x)ρ\frac{\rm{d}\rho}{\rm{d}t} = \partial_t \rho + \frac{\rm{d}x}{\rm{d}t}\partial_x \rho =\partial_t \rho + v(x)\partial_x\rho= -v'(x)\rho dtdρ​=∂t​ρ+dtdx​∂x​ρ=∂t​ρ+v(x)∂x​ρ=−v′(x)ρ

比起其他妖魔鬼怪，这个方程实在是太好解了！
将方程改写成我们熟悉的形式

dρρ=−v′(x)dt\frac{\rm{d}\rho}{\rho} = -v'(x)\rm{d}t ρdρ​=−v′(x)dt

两边积分

ln⁡ρ(x(t,x0),t)−ln⁡ρ0(x0)=−∫0tv′(x)dτ\ln \rho(x(t,x_0),t) - \ln \rho_0(x_0) = -\int^t_0 v'(x) \rm{d}\tau lnρ(x(t,x0​),t)−lnρ0​(x0​)=−∫0t​v′(x)dτ

这里有几点需要注意的，首先是我们代入了ρ(x,0)=ρ0(x)\rho(x,0)=\rho_0(x)ρ(x,0)=ρ0​(x)的初值条件
其次稍微换了一下符号，因为这里我们是对[0,t][0,t][0,t]区间积分，把原来的ttt换成了τ\tauτ
继续变换（使用了dx/dτ=v(x(τ))\rm dx/d\tau = v(x(\tau))dx/dτ=v(x(τ))）

−RHS=∫0tv′(x(τ))dτ=∫0tv′(x(τ))v(x(τ))dxdτdτ=∫x0x(t,x0)v′(x)v(x)dx=ln⁡v(x(t,x0))v(x0)-RHS = \int^t_0 v'(x(\tau))\rm{d}\tau = \int^t_0 \frac{v'(x(\tau))}{v(x(\tau))}\frac{\rm{d}x}{\rm{d}\tau} \rm{d}\tau = \int^{x(t,x_0)}_{x_0} \frac{v'(x)}{v(x)}\rm{d}x = \ln \frac{v(x(t,x_0))}{v(x_0)} −RHS=∫0t​v′(x(τ))dτ=∫0t​v(x(τ))v′(x(τ))​dτdx​dτ=∫x0​x(t,x0​)​v(x)v′(x)​dx=lnv(x0​)v(x(t,x0​))​

整理一下就是

ρ(x(t,x0),t)=ρ0(x0)v(x0)v(x(t,x0))\rho(x(t,x_0),t) = \rho_0(x_0)\frac{v(x_0)}{v(x(t,x_0))} ρ(x(t,x0​),t)=ρ0​(x0​)v(x(t,x0​))v(x0​)​

接下来还有一点问题，其实刚刚的x0x_0x0​是任意的，现在我们需要根据确定的ttt来反推x0x_0x0​。这个事情通过求解特征线方程来解决，我们先求dx/dt=v(x(t)),x(0)=x0\rm dx/dt = v(x(t)),x(0)=x_0dx/dt=v(x(t)),x(0)=x0​的解为x=x(t,x0)x=x(t,x_0)x=x(t,x0​)，然后反推就能得到x0=φ(x,t)x_0=\varphi(x,t)x0​=φ(x,t)
最终的结果是

ρ(x,t)=ρ0(φ(x,t))v(φ(x,t))v(x)\rho(x,t) = \rho_0(\varphi(x,t))\frac{v(\varphi(x,t))}{v(x)} ρ(x,t)=ρ0​(φ(x,t))v(x)v(φ(x,t))​
